# Cat Cafe

A game where you are the waitress on a Cat Cafe and have to deliver desserts to impatient cats.

![cat cafe title](./screenshot.png)

# Developer Notes

- `npm install -g cordova`
- `npm install`

## Running the local browser

- `npm run build`
- `cordova prepare`
- `cordova run browser`
